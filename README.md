### O que eu aprendi

Neste projeto foi melhorado as habilidades de css, tendo ênfase na propriedade flex, Footer e em como deixar uma página responsiva.

``` css
@media screen and (max-width :780px){
    .conteiner{
        flex-direction: column;
    }
}
```
